# Responsive Portfolio

A modern, responsive personal portfolio website built with **React.js**, **Tailwind CSS**, and **Framer Motion**.  
This project showcases projects, skills, and experience with smooth animations and a clean UI.

## 🚀 Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth animations using **Framer Motion**
- Modern UI styled with **Tailwind CSS**
- Icon support via **React Icons** and **Lucide React**
- Reusable and clean React components
- Fast performance and optimized build

## 🛠️ Tech Stack

- **React.js** – Frontend framework
- **Tailwind CSS** – Utility-first CSS framework
- **Framer Motion** – Animations and transitions
- **React Icons** – Popular icon packs
- **Lucide React** – Clean and customizable SVG icons


## ⚙️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Tosif2230/Portfolio.git

2. Navigate to the project directory:
   cd responsive-portfolio

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

The app will run locally at:
http://localhost:5173

📦 Build for Production
    npm run build

🎨 Customization

Update personal information in the components under src/pages

Replace images in the src/assets folder

Modify colors, fonts, and spacing via tailwind.config.js

Add or remove sections as needed
